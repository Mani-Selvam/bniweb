# bniweb

A React + Vite single-page application.

## Tech Stack
- React 19
- Vite 8
- ESLint

## Project Structure
- `src/` - React source code (App.jsx, main.jsx, styles, assets)
- `public/` - Static public assets (favicon, icons)
- `index.html` - HTML entry point
- `vite.config.js` - Vite configuration (configured for Replit: host 0.0.0.0, port 5000, allowedHosts true)

## Development
The dev server runs via the `Start application` workflow using `npm run dev` on port 5000.

## Deployment
Configured as a static deployment:
- Build: `npm run build`
- Public directory: `dist`
